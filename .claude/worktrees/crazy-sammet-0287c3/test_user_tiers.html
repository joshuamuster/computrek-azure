User Tier Testing

This is a simple test to verify the user authentication system works.

Test Instructions:

  1. Run the development server: npm run dev
  2. Open the browser console and run these commands to test different user tiers:
    • Cadet: localStorage.setItem('userRole', 'cadet'); localStorage.setItem('isAuthenticated', 'true'); location.reload();
    • Parent: localStorage.setItem('userRole', 'parent'); localStorage.setItem('isAuthenticated', 'true'); location.reload();
    • Admin: localStorage.setItem('userRole', 'admin'); localStorage.setItem('isAuthenticated', 'true'); location.reload();
  3. Navigate to these pages to verify different messages appear:
    • /courses (CoursesMain.vue)
    • /grades (Grades.vue)
    • /crew (Crew.vue)

Expected Results:

CoursesMain.vue (/courses):

  • Cadet: "Computer Science Discoveries is an introductory mission..."
  • Parent: "Welcome to your child's Computer Science Discoveries journey..."
  • Admin: "Administrative overview of Computer Science Discoveries curriculum..."

Grades.vue (/grades):

  • Cadet: "Track your academic performance across all class periods..."
  • Parent: "Monitor your child's academic progress..."
  • Admin: "Administrative grade overview displaying student performance analytics..."

Crew.vue (/crew):

  • Cadet: "Meet your instructors and support crew..." & "Your technical instructors are here to help..."
  • Parent: "Get to know the teaching staff..." & "Our technical teaching staff specializes..."
  • Admin: "Administrative staff directory..." & "Technical staff assignments..."
